## Parancsok:

`!hello`- A súgó megjelenítése. \
`!hello setchannel #csatorna`- Csatorna Beállítása. \
`!hello setmessage <üzenet>` - Üzenet beállítása. \
`!hello toggle` - Ki/Be kapcsolása. \
`!hello setrole @rang` - Automatikus rang beállítása. \
`!hello test` - Üzenet tesztelése. \
`!hello info` - Beállítások megjelenítése.

`!welcome` - A súgó megjelenítése. \
`!welcome setchannel #csatorna `- Csatorna Beállítása. \
`!welcome setmessage <üzenet>` - Üzenet beállítása. \
`!welcome toggle` - Ki/Be kapcsolása. \
`!welcome setrole @rang` - Autmoatikus rang beállítása. \
`!welcome test` - Üzenet tesztelése. \
`!welcome info` - Beállítások megjelenítése.

# Változók:

`{user}` - Felhasználó Globáis Neve. \
`{username}` - Felhasználó neve. \
`{server}` - Szerver neve. \
`{membercount}` - Tagok száma. \
`{membercount_ordinal}` - Tagok száma az angol számazás szerint. (PL: 1st, 2rd) \
`{inviter}` - Aki meghívta a felhasználó-t a szerverre.
